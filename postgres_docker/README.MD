# Запуск проекта в Docker

## Доступ к Postgres:

- **URL:** `postgres:5432`
- **Username:** `postgres`
- **Password:** `changeme`

## Доступ к PgAdmin:

- **URL:** `http://localhost:5050`
- **E-mail:** `pgadmin@example.com`
- **Password:** `admin`

Запустить контейнер:
```dockerfile
docker-compose up -d
```
